# Super Coach Step-by-Step Tennis Website

Open `index.html` in a browser to run the step-by-step demo.

Pages:
1. `index.html` — profile setup with age, height, gender, hand, and live character preview.
2. `player-style.html` — ATP/WTA-style top 10 player model selection.
3. `devices.html` — camera and smart-watch toggles, then a 3-2-1 countdown and training recording screen.
4. `report.html` — final performance report with one-click PDF download.

Version 9 updates:
- Uses the uploaded Super Coach logo and favicon with transparent backgrounds.
- Dark mode shows the logo without a white box; light mode keeps a clean white logo panel.
- PDF export is now fully local/offline. It captures the visible report area as an image-based PDF instead of loading external libraries or opening the print page.
